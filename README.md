# AI Resume Analyzer Pro

AI-powered resume analysis tool built using Python and Flask.

## Features

- Resume PDF Upload
- PDF Text Extraction
- Skill Extraction Engine
- Job Description Matching
- ATS Score Calculation
- Recruiter Dashboard

## Tech Stack

- Python
- Flask
- PyPDF2
- HTML
- CSS
- JavaScript
- Scikit Learn

## Workflow

Upload Resume → Extract Text → Detect Skills → Match JD → Calculate ATS Score
